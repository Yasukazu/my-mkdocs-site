# my-mkdocs-site

a personal MkDocs site

## Installs

1. MkDocs

### Static site generator itself

```bash
pip install mkdocs
```

2. Material theme

<details>
  <summary> a theme with useful intrinsic plugins</summary>
 - search
 - blog
 - tags
 </details>

```bash
pip install mkdocs-material
```

3. glightbox

image sizing and alignment

```bash
pip install mkdocs-glightbox
```
